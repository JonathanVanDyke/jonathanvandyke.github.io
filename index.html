Thingyverse

Fullstack clone of the thingiverse 3D printing site

A clone of the popular 3D printing site, Thingiverse! This site was built as a single-page app using a React / Redux frontend with a Ruby on Rails Backend. Create, like, and collect yours and other's 3D prints! This code features modular print and collection cards that have a standarized input. This standard input can be used anywhere on the site through a flat front-end state to create a filtered list of components!
